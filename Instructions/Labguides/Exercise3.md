# Exercise 3: Use an agent of your choice

### Estimated Duration: 20 Minutes

## Lab Overview

In this exercise, you will explore how different Microsoft 365 Copilot agents can be used together to support decision-making and strategic planning for Project Nexus. While previous exercises focused on analyzing data and gathering insights, this lab focuses on transforming those insights into executive-ready communication and forward-looking strategies. You will use the **Writing Coach** agent to generate structured executive reports, emails, and briefing documents, and the **Idea Coach** agent to brainstorm enterprise rollout strategies and adoption improvements. By the end of the exercise, you will see how multiple Copilot agents can assist with producing leadership-ready outputs and planning the next phase of a project rollout.

# Lab Objectives

In this exercise, you will complete the following tasks:

- Task 1: Generate Executive Communications Using Writing Coach

- Task 2: Brainstorm Rollout Strategies Using Idea Coach

## Task 1: Generate Executive Communications Using Writing Coach

In this task, you will add the Writing Coach agent and use it to generate executive-level content for Project Nexus, including recommendation reports, executive emails, mitigation plans, and leadership-ready briefing documents.

1. You are still on the **Microsoft 365** page. From the left navigation pane, select **All agents** under Agents section.

    ![](./media/ex3-p4t1p1.png)

1. Scroll down to the **Built by Microsoft** section and select **See more** to view additional available agents.

    ![](./media/ex3-p4t1p2.png)

1. From the **Built by Microsoft** section, select the **Writing Coach** agent.

    ![](./media/ex3-p4t1p3.png)

1. On the **Writing Coach** agent page, select **Add** to add the agent to your Microsoft 365 Copilot environment.

    ![](./media/ex3-p4t1p4.png)

1. From the left navigation pane under **Agents**, select **Writing Coach** to open the agent.

    ![](./media/ex3-p4t1p5.png)

1. In the prompt field **(1)**, enter the following prompt and then click the **Send (2)** arrow to submit the request:

    ```
    Draft a structured executive recommendation report for Project Nexus including adoption performance, financial alignment, risks, and a clear go or no-go recommendation.
    ```

    ![](./media/ex3-p4t1p6.png)

1. Review the response generated by the **Writing Coach** agent, which provides a structured executive recommendation report for **Project Nexus** including adoption performance, risks, financial alignment, and a go/no-go recommendation.

    ![](./media/ex3-p4t1p7.png)

    > **Note:** The responses generated by Copilot agents may vary depending on the prompt interpretation and the available data. As a result, the output you receive may differ from the examples shown in the images.
    
1. Enter the following prompt and review the updated response generated by the Writing Coach agent:


    ![](./media/ex3-p4t1p8.png)

    > **Note:** The responses generated by Copilot agents may vary depending on the prompt interpretation and the available data. As a result, the output you receive may differ from the examples shown in the images.

1. Enter the following prompt and review the response:

    ```
    Draft a concise executive email summarizing Project Nexus outcomes and recommended next steps.
    ```

    ![](./media/ex3-p4t1p9.png)

1. Now enter the following prompt and review the response:

    ![](./media/ex3-p4t1p10.png)

    > **Note:** The responses generated by Copilot agents may vary depending on the prompt interpretation and the available data. As a result, the output you receive may differ from the examples shown in the images.

## Task 2: Brainstorm Rollout Strategies Using Idea Coach

In this task, you will install and use the Idea Coach agent to brainstorm strategies that support the successful enterprise rollout of Project Nexus, including adoption improvement ideas, HR engagement strategies, and a phased rollout plan.

1. Select **All agents (1)** from the left navigation pane, then scroll to the **Built by Microsoft** section and click **See more (2)** to view additional available agents.

    ![](./media/ex3-p4t1p11.png)

1. Select the **Idea Coach** agent from the **Built by Microsoft** section to open its details and explore how it can assist with brainstorming and idea development.

    ![](./media/ex3-p4t1p12.png)

1. Select **Add** to install the **Idea Coach** agent and make it available within Microsoft 365 Copilot.

    ![](./media/ex3-p4t1p13.png)

1. From the left navigation pane under **Agents**, select the **Idea Coach** agent to open it and begin interacting with it in Microsoft 365 Copilot.

    ![](./media/ex3-p4t1p14.png)

1. In the prompt field, enter the following prompt and review the response generated by the **Idea Coach** agent:

    ```
    Brainstorm strategies to ensure successful enterprise-wide rollout of Project Nexus despite varying adoption levels across departments.
    ```

    ![](./media/ex3-p4t1p15.png)

    > **Note:** The responses generated by Copilot agents may vary depending on the prompt interpretation and the available data. As a result, the output you receive may differ from the examples shown in the images.

1. Enter the following prompt in the **Idea Coach** prompt field and review the response generated:

    ```
    Generate innovative ideas to improve HR adoption and reduce reliance on email-based coordination.
    ```

    ![](./media/ex3-p4t1p16.png)

1. Enter the following prompt in the **Idea Coach** prompt field and review the response generated:

    ```
    Outline a phased enterprise rollout plan for Project Nexus including training, communication, governance, and performance tracking.
    ```

    ![](./media/ex3-p4t1p17.png)

    > **Note:** The responses generated by Copilot agents may vary depending on the prompt interpretation and the available data. As a result, the output you receive may differ from the examples shown in the images.

1. Use this opportunity to explore some of the other **prebuilt agents**. Review the sample prompts provided for each agent to understand the types of tasks they are designed to handle. You can run the suggested prompts or enter your own custom prompts to see how different agents assist with various requests.
